# JavHub 味道补完 · 内容层去玻璃 Patch

> 唯一漏掉的结构层是**内容大容器仍是玻璃**。改这一刀，图底关系立刻清晰，海报成为唯一的颜色与深度。

## 改动 1 ★核心★ — `.main-content` 从玻璃改实底
**文件：** `frontend/src/App.vue`（`<style>` 内，约 811 行）

**Find：**
```css
.main-content {
  flex: 1;
  overflow-y: auto;
  overflow-x: hidden;
  overscroll-behavior: contain;
  scrollbar-gutter: stable;
  -webkit-overflow-scrolling: touch;
  scroll-padding-block: 30px var(--app-chrome-inset);
  min-width: 0;
  margin: var(--app-chrome-inset);
  background:
    var(--surface-specular-edge),
    var(--surface-noise),
    var(--content-material);
  border: 1px solid var(--content-material-border);
  border-radius: var(--radius-sheet);
  box-shadow: var(--glass-surface-shadow);
  position: relative;
  z-index: var(--z-raised);
}
```

**Replace：**
```css
.main-content {
  flex: 1;
  overflow-y: auto;
  overflow-x: hidden;
  overscroll-behavior: contain;
  scrollbar-gutter: stable;
  -webkit-overflow-scrolling: touch;
  scroll-padding-block: 30px var(--app-chrome-inset);
  min-width: 0;
  margin: var(--app-chrome-inset);
  /* 内容区 = 不透明窗体底色。玻璃只留给 chrome（侧栏/底栏/工具栏）。
     这样海报与封面成为画面里唯一的颜色与深度，图底关系清晰。 */
  background: var(--canvas);
  border: 1px solid var(--hairline);
  border-radius: var(--radius-sheet);
  box-shadow: var(--shadow-card);
  position: relative;
  z-index: var(--z-raised);
}
```

同一文件下方 `.main-content:focus-visible` 里的 `box-shadow: var(--glass-surface-shadow), var(--focus-ring);`
顺手改成 `box-shadow: var(--shadow-card), var(--focus-ring);`（保持阴影一致，非必须）。

## 改动 3 — 默认主题设为深色（口味向，建议）
**文件：** `frontend/src/assets/themes.js`（约 5 行）

这套 poster-first / 色彩从内容来的语言在**深色**下才封神（封面色彩张力拉满）；浅色下海报浮在白底偏平淡。
让首次进入就是强的那一版：

**Find：**
```js
export const DEFAULT_THEME_KEY = 'apple-light'
```
**Replace：**
```js
export const DEFAULT_THEME_KEY = 'apple-dark'
```

注意点：
- 只影响**没手动选过主题**的用户（localStorage 为空时）。已选过的用户保留各自选择，零影响。
- 浅色主题完整保留，切换按钮照常用。
- 若有测试（如 `themes` 相关 spec 或 `restoreTheme`）硬断言了 `DEFAULT_THEME_KEY === 'apple-light'` 或默认返回 light，一并改成 dark。grep `apple-light` 与 `DEFAULT_THEME_KEY` 自查一遍。
- 不想动全局默认也行——退而求其次：在你自己的 localStorage 里把 `javhub_theme` 设成 `apple-dark` 即可，仅影响你本机。

## 关于"调淡背景纹理"——撤回，别动 ⚠️
我原本列了第 4 条"调淡 `--app-backdrop-texture`"，**深看代码后撤回**，给你讲清逻辑：

`--app-backdrop-texture` 铺在 `.app-layout::before`（最底层）。之前觉得它"浑浊"，其实浑浊来自**内容玻璃面板叠在它上面**——改动 1 之后，不透明的 `.main-content` 会把这层纹理**完全盖住**，纹理只剩两处可见：
1. 内容面板四周 12px 的 inset 缝隙（很淡，是质感，留着好看）；
2. **透过玻璃侧栏 / 底栏折射出来**——而这正是玻璃之所以"像玻璃"的关键。玻璃需要背后有东西可折射，没纹理玻璃就成了死板的半透明色块。

所以这层纹理是**给 chrome 玻璃供养的**，是 load-bearing 的，**保持不动**。改动 1 一落地，"浑浊感"自己就消失了。
