JavHub · Design Language v2

从"精致的玻璃"到"内容当主角"

这份文档记录重设计背后的判断与系统，方便照着改回现有 Vue 代码。核心不是推翻你的 Liquid Glass，而是把它放回正确的位置。

四条核心修正

01

玻璃只给 chrome，内容区改实底

原方案侧栏、内容、卡片全是半透明玻璃，导致没有图/底层次。新方案：侧栏 / 顶栏 / 浮动工具栏 = 玻璃（blur + saturate），内容画布 / 卡片 = 实底（--canvas / --card）。这样海报才能跳出来。

02

Poster-first，色彩从内容来

影片库的视觉锚点是封面。卡片 3:4 大图、edge-to-edge，UI 的彩色全部由封面提供。深色底让这种色彩张力最大化——这也是你那张策展稿的方向。

03

引入克制的 accent tint

原方案 --accent: #1D1D1F（纯黑），全站灰阶 → 像线框。新方案给一个真正的强调色（默认靛蓝），只用于：主按钮、激活态、进度、关键数字。封面色不抢，accent 才克制。

04

用起 display 字阶，标题更自信

原方案 body 14px、满屏 11–12px，密而不分层。新方案 hero 用 40px display、区块标题 22–30px，层级靠"字号 × 字重 × 色阶"三轴拉开。

颜色

深色为主（封面色彩最突出），浅色完整兜底。背景不是纯黑，带极轻的冷调与 accent 辉光。

间距 · 圆角 · 材质

  • 间距：严格 4/8 阶梯 --s1…--s10（4·8·12·16·20·24·32·44·56·72），杜绝 11/13/19 这种"差一点"。
  • 圆角：海报 14px · 卡片 18px · 大面板/玻璃 24px · 控件胶囊 999px。
  • 玻璃材质：仅 chrome 用 backdrop-filter: blur(40px) saturate(1.6) + 1px specular 高光边。内容卡用实底 + --shadow-card。
  • 动效：标准 --ease，微弹 --spring 仅给浮岛/弹窗/开关。全局 prefers-reduced-motion 兜底。

组件清单

  • PosterCard — 全站主角。封面 + 番号/类型/时长角标 + hover 收藏 + 可选演员订阅条 + 可选多选态。
  • ActRow（演员订阅条）— 翻译名优先、原名辅助、绿=已订阅/灰=可订阅，对齐你的策展稿。
  • StatCard / Panel — 仪表盘统一卡面。
  • 状态点 + Badge — ok/warn/bad/info 四档，脉冲圈表示在线。
  • 浮动玻璃工具栏（batch-bar）— 多选时从底部弹出，承载批量操作。
  • iOS Switch / Segmented — 设置与筛选统一控件。
  • VideoModal — 沉浸式详情：大封面头图 + ←/→ 翻片 + ESC 关闭 + 相关推荐轨道。

关键页面：改了什么

改之前

落地页 = 「下载管理」，一进门就是任务队列。

改之后

落地页 = 「今日」仪表盘：继续观看 → KPI → 待处理行动中心 → 下载中 → 订阅更新。

改之前

候选/缺磁力/翻译散落在各页，要逐个去找。

改之后

首页「待处理」把它们收口成一个行动中心；候选确认页支持批量审核。

改之前

收藏与影库是普通灰阶列表。

改之后

poster-first 网格 + 私人策展（卡片下挂演员订阅）+ 影库批量多选。
